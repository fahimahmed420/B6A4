# B6A4

## MediStore — Online Pharmacy Frontend

A full-stack e-commerce frontend for purchasing over-the-counter medicines, built with **Next.js 15**, **TypeScript**, and **Tailwind CSS**.

## Tech Stack
- **Framework**: Next.js 15 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **HTTP Client**: Axios
- **Notifications**: React Hot Toast
- **Icons**: Lucide React

## Features

### Customer
- Browse medicines with search, category filter, and price range
- View detailed medicine pages with reviews
- Shopping cart with quantity management
- Checkout with Cash on Delivery
- Order tracking and cancellation
- Write reviews with star ratings
- Profile management

### Seller
- Dashboard with inventory and order management
- Add, edit, delete medicines
- Update order statuses

### Admin
- Platform statistics overview
- User management with ban/unban
- All orders view
- Category management

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000)
